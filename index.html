WHAT THIS MEANS FOR YOU

개인 실험 vs 프로덕션

개인 실험 · 자동화
  • 크레딧 범위 ($20~$200) 내 충분
  • OAuth 방식 유효 — 로컬/개인용
프로덕션 · 대규모
  • 공식 문서가 API Key 종량제 권장
  • 크레딧은 개인 단위 — 풀링 불가
정석 워크플로우 — CLI로 프로토타입 → SDK로 프로덕션. 단 6/15 이후, “완전 대체”하면 과금 버킷이 바뀐다.
